# Skills 文档索引

本目录包含用于提升 AI 助手效率的 Skills 文档。

## 📚 现有 Skills

### 项目文档导航
- **project-documentation.md** - 项目文档索引和使用指南
  - 快速定位项目中的 README 文档
  - 提供文档使用策略和最佳实践
  - 帮助 AI 优化对话上下文

### 架构和设计
- **project-architecture.md** - 项目架构说明
- **shape-rendering.md** - 形状渲染技术
- **vml-rendering.md** - VML 渲染处理
- **css-styling.md** - CSS 样式管理
- **list-numbering.md** - 列表编号系统

## 🎯 使用建议

在对话开始时，AI 应该：
1. 阅读 `project-documentation.md` 了解文档体系
2. 根据任务类型定位相关的 README 文档
3. 阅读相关 README 获取详细的模块信息
4. 参考其他 skills 了解特定技术细节
