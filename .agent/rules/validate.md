# 验证步骤

1. 第一步：工作空间内运行 `pnpm run build`
2. 第二步：工作空间内运行 `pnpm i`
3. 第三步：工作空间内运行 `pnpm run pg:dev`
4. 第四步：检查浏览器渲染效果，对比截图
5. 第五步：审查元素“图片 1、图片 2、图片 3”显示效果，与用户提供的截图进行比较，像素级对比
6. 第六步：审查元素“矩形 4、矩形 5”显示效果，与用户提供的截图进行比较，像素级对比
7. 第七步：审查元素“肘型连接符 8、肘型连接符 9”显示效果，与用户提供的截图进行比较，像素级对比
8. 第八步：审查元素“曲线连接符 10”显示效果，与用户提供的截图进行比较，像素级对比
9. 第九步：审查元素“任意多边形 11”显示效果，与用户提供的截图进行比较，像素级对比
