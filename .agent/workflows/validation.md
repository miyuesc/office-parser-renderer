---
description: 验证步骤
---

- 第一步：工作空间内运行 `pnpm run build`
- 第二步：工作空间内运行 `pnpm i`
- 第三步：工作空间内运行 `pnpm run pg:dev`
- 第四步：检查浏览器渲染效果，对比截图
- 第五步：审查元素“图片 1、图片 2”显示效果，与用户提供的截图进行比较，像素级对比
- 第六步：审查元素“矩形 4、矩形 5”显示效果，与用户提供的截图进行比较，像素级对比
- 第七步：审查元素“肘型连接符 8、肘型连接符 9”显示效果，与用户提供的截图进行比较，像素级对比
- 第八步：审查元素“曲线连接符 10”显示效果，与用户提供的截图进行比较，像素级对比
- 第九步：审查元素“任意多边形 11”显示效果，与用户提供的截图进行比较，像素级对比
- 第十步：审查七角星、五边形、不等式符号元素显示效果，与用户提供的截图进行比较，像素级对比
- 第十一步：审查元素图表显示效果，与用户提供的截图进行比较，像素级对比
