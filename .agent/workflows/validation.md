---
description: 验证步骤
---

1. 第一步：工作空间内运行 `pnpm run build`
2. 第二步：工作空间内运行 `pnpm i`
3. 第三步：工作空间内运行 `pnpm run pg:dev`
4. 第四步：检查浏览器渲染效果，与用户提供的截图进行比较，像素级对比
5. 第五步：审查图片元素显示效果，与用户提供的截图进行比较，像素级对比
6. 第六步：审查形状元素显示效果，与用户提供的截图进行比较，像素级对比
7. 第七步：审查连接符元素显示效果，与用户提供的截图进行比较，像素级对比
8. 第八步：审查任意多边形元素，与用户提供的截图进行比较，像素级对比
9. 第九步：审查所有图表显示效果，与用户提供的截图进行比较，像素级对比
10. 第十步：审查界面布局效果（间距、分页、分栏等），与用户提供的截图进行比较
